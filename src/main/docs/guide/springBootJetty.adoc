As shown in the previous app engine configuration file, we are using Jetty.

Grails is built on top of SpringBoot. Following SpringBootâ€™s documentation, we need to do the following changes to https://docs.spring.io/spring-boot/docs/current/reference/html/howto-embedded-servlet-containers.html[deploy to Jetty instead of Tomcat].

Replace:

[source, groovy]
.build.gradle
----
compile "org.springframework.boot:spring-boot-starter-tomcat"
----

with:

[source, groovy]
.build.gradle
----
provided "org.springframework.boot:spring-boot-starter-jetty"
----

NOTE: it is important you set the `spring-boot-starter-jetty` dependency as `provided`.
